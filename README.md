# 栈
栈是一种特殊的线性表，仅能在线性表的一端操作，栈顶允许操作，栈底不允许操作。<br>
栈的特性：后进先出<br>

# 队列
队列是一种特殊的线性表<br>
特殊之处在于它只允许在表的前端（front）进行删除操作，而在表的后端（rear）进行插入操作<br>
和栈一样，队列是一种操作受限制的线性表。进行插入操作的端称为队尾，进行删除操作的端称为队头。
